✅ INTEGRAÇÃO NOTION
   • Biblioteca oficial: notion-client
   • Registra em thread separada (não bloqueia GUI)
   • Payload EXATO com 4 propriedades:

     - "Inicio" (Date): timestamp de início
     - "Fim" (Date): timestamp de término
     - "Tecnologia" (Select): categoria selecionada
   • Feedback visual: sucesso ou erro

   • Classe NotionService: gerencia API
   • Classe PomodoroTimer: lógica do cronômetro
   • Classe PomodoroApp: interface e coordenação
   • Métodos privados (_) e públicos
   • Type hints em todas as funções

🚀 INÍCIO RÁPIDO:
═════════════════════════════════════════════════════════════════════════════════

1. INSTALAR DEPENDÊNCIAS:
   └─ pip install -r requirements.txt

2. CONFIGURAR NOTION:
   └─ Acesse: https://www.notion.so/my-integrations
   └─ Crie integração e copie token
   └─ Crie database com 4 propriedades (ver NOTION_SETUP.md)
   └─ Copie DATABASE_ID da URL

3. CONFIGURAR .ENV:
   └─ cp .env.example .env
   └─ Edite com NOTION_API_KEY e DATABASE_ID

4. EXECUTAR:
   └─ python main.py

5. TESTAR (opcional):
   └─ python tests.py (menu interativo)

💻 FLUXO DE USO:
═════════════════════════════════════════════════════════════════════════════════

  1. Execute: python main.py
  2. Digite a tarefa (ex: "Implementar login")
  3. Selecione a categoria (ex: "Python")
  4. Clique [Iniciar]
  5. Timer começa: 25:00 → 24:59 → ... → 00:00
  6. Ao chegar em 00:00:
     └─ Timer para
     └─ Envia automaticamente para Notion em thread separada
     └─ Label mostra: "✓ Sucesso!" ou "✗ Erro"
  7. Clique [Resetar] para nova sessão
